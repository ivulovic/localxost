Function that creates height in Xvh where X is number from 1 to 10.

Example: row-10-height. Row will have 100vh height.

@for $i from 1 through 10{
  .row-#{$i}-height{
    height: #{$i * 10}vh;
  }
}

Function that creates no-padding-X classes where X represents side of element.

$sides: (top, bottom, left, right);

@each $name in $sides {
  .no-padding-#{$name} {
    padding-#{$name}: 0px;
  }
  .no-margin-#{$name} {
    margin-#{$name}: 0px;
  }
}
